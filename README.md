# Universal Unit Converter

A robust, feature-rich desktop application built with **Python 3** and **PyQt6** that handles standard structural conversions (Distance, Weight, Volume, Time), specialized mathematical logic (Temperature), and real-time financial tracking (Live Currency Conversion).

This project was built using professional software engineering best practices, focusing on modular architecture, responsive UI design, and automated testing.

## Key Features
* **Multi-Category Conversion:** Seamlessly switch between Distance, Weight, Volume, Time, and Temperature.
* **Dynamic & Responsive UI:** Built using PyQt6 layout managers to ensure fluid UI scaling and resizing across different desktop resolutions.
* **Extensible Architecture:** Strict separation of concerns—UI layout and core mathematical conversion logic are completely decoupled.
* **Live Currency Exchange (Upcoming):** Integration with a external REST API to fetch up-to-the-minute global exchange rates.
* **Automated Unit Testing:** Full test coverage for core conversion engines to ensure mathematical accuracy and prevent regressions.
